# free-wireframe-tool

📌 The Basics
Add Elements: Drag and drop icons from the left toolbar onto the canvas.
Select & Move: Click an element to select it. Click and drag to move it around.
Edit Text: Double-click on any text within a component to edit it. Click outside to save.
Resize: Select an element and drag the circular handles on its corners or edges. Hold Shift to scale proportionally.⚡ Connectors & Flowcharts
Draw Lines: Hover over a selected element to reveal blue connection nodes. Click and drag from a node to another element to draw a connector line.
Edit Lines: Click a line to select it. Use the Properties Panel on the right to change it to Straight, Curved, or Orthogonal (Elbow), and add arrows or dots to the ends.

🎨 Properties Panel (Right Side)
Formatting: Change width, height, font size, bold, italic, underline, or strikethrough.
Component States: Change an element to look Disabled, Focus, Active, or Read-only using the State dropdown.
Layering: Use the Layer arrows to send elements to the front or back.
Smart Components: If you select a Dropdown, Tabs, or Table, special text boxes will appear allowing you to quickly edit options (comma-separated).

⌨️ Keyboard Shortcuts & Canvas Controls
Middle Click or Hold Spacebar + Drag to pan across the infinite canvas.
Scroll Wheel to Zoom In and Out.
Ctrl + C / Ctrl + V to Copy and Paste selected elements.
Ctrl + V to paste Images or SVGs directly from your clipboard onto the canvas.
Ctrl + Z / Ctrl + Shift + Z to Undo / Redo.
Delete or Backspace to remove selected elements.
Arrow Keys to nudge elements. Hold Shift to nudge by 10 pixels.
Ctrl + S to Quick Save to your browser's local storage.

💾 Saving & Exporting
Save JSON: Downloads your raw project file.
Load JSON: Restores a saved project. You can also directly Drag & Drop a JSON file onto the canvas to load it instantly.
Export PNG: Select the elements you want to export, then click Export to PNG to download a clean image file of your selection.
